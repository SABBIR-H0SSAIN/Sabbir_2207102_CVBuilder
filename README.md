# Sabbir - 2207102 CV Buider
This is a simple CV Builder app made with JavaFX. You just have to enter your details and the app will build a clean CV for you.The app contains clean and beautiful user interface
## Folder Structure:
![Folder structure](screenshots/img.png)

## Features:
- Beautiuful user Interface
- Clear navigations between each pages 
- Image upload for your cv
- Validation on Input fields
- Displays alert dialog in each validation and confirmations
- User can edit the generated CV again
- Followed MVC pattern
