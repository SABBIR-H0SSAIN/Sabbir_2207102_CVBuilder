# Sabbir - 2207102 CV Buider
This is a simple CV Builder app made with JavaFX. You just have to enter your details and the app will build a clean CV for you.

## Features:
- Beautiuful user Interface
- Clear navigations between each pages 
- Image upload for your cv
- Validation on Input fields
- Displays alert dialog in each validation and confirmations
- Store Generated cv to local database using sql lite database
- User can edit, delete or update the generated CV from database
- Used threads for running tasks in background to boost app performance
- Followed MVC pattern
